# 使用模板项目流程

## 1. 复制模板

将 `D:\git-projects\uniappx-app-template` 复制到你的目标项目目录。

## 2. 初始化项目标识

优先修改以下文件：

1. `manifest.json`
2. `.env.development`
3. `.env.tunnel`
4. `.env.production`
5. `lib/config.uts`
6. `pages/login/index.uvue`
7. `pages/home/components/medicine-list-tab.uvue`
8. `pages/home/components/profile-tab.uvue`
9. `AGENTS.md`
10. `README.md`

## 3. 清理示例业务

默认示例业务是药品管理 App，请按你的业务替换：

- 页面标题
- 列表字段
- 登录协议文案
- 个人页统计项
- 上传字段与接口路径

## 4. 自检

按你的实际工程方式执行最基本自检：

1. 在 HBuilderX 中打开项目
2. 运行到目标平台
3. 检查登录页、首页、录入页、个人页是否正常跳转
4. 检查接口地址是否指向当前环境

## 5. 开始业务开发

完成以上步骤后，再进入实际业务开发。
